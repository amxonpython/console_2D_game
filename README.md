# Console Game Engine
Консольная RPG-бродилка на Java. Учебный проект.

## Управление
- WASD — движение
- E — инвентарь (временно не работает)
- exit — выход (временно не работает)

## Механики
- Зомби преследует игрока
- Предметы на карте (+ бинт)
- Инвентарь с слотами (временно не работает)
- HP и смерть (частично не работает)

для запуска просто запустите Main в любом редакторе кода
