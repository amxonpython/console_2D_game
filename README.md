# Simple Java Roguelike Engine 🎮

Экспериментальный игровой движок на Java для создания пошаговых/тиковых текстовых игр (Roguelike) с графическим интерфейсом на базе **Java Swing** и многопоточной архитектурой игрового цикла.

## 🚀 Особенности проекта

* **Игровой цикл (Game Loop):** Выделенный фоновый поток (`Thread`) для обработки игровых тиков, логики и отрисовки независимо от интерфейса.
* **Событийно-ориентированное управление:** Управление персонажем перенесено с консольного `Scanner` на графические кнопки Swing.
* **Механика сетки (Matrix):** Динамическая генерация двумерного игрового поля $10 \times 10$.
* **Система NPC:** Базовая логика преследования игрока врагами (Зомби).
* **Инвентарь и предметы:** Возможность подбирать предметы (например, бинты) на карте, сохранять их в слоты инвентаря и использовать для восстановления характеристик (HP).

## 📂 Структура проекта

```text
src/
│
├── NPS/
│   └── zombie/
│       └── zombie.java          # Логика поведения, спавна и атаки противников
│
├── objects/
│   ├── bandage/
│   │   └── bandage.java         # Класс предмета "Бинт", логика поднятия и лечения
│   └── data/
│       └── using_objects.java   # Глобальный обработчик использования предметов по ID
│
├── player/
│   ├── inventory.java           # Управление слотами инвентаря персонажа
│   ├── parameters_player.java   # Характеристики игрока (HP, голод, скорость)
│   └── player.java              # Логика перемещения игрока по координатам
│
├── world/
│   ├── matrix.java              # Двумерный массив карты, методы заполнения и вывода
│   └── World.java               # Главный игровой цикл, управление потоками и тиками
│
└── Main.java                    # Точка входа, инициализация графического интерфейса Swing
